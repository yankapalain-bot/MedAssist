# MedAssist
Lightweight medical assistant

## Description
Clinic Assistant App to help in A specific Gynecology Africa context 

## Ojectives 

- Centralizes patient records 
- Improves clinical follow-up tracking 
- Reduces missed appointments 
- Enhances daily productivity for physicians 
- The application is not a full hospital system, but a focused tool designed for practical daily use.

## Technologies  

### Django | HTMX | DaisyUI | Bokeh | Folium




## Core features


###  1- Patient Management

Create and manage patient profiles 
Store basic medical information 
Quick search and access 


### 2- Clinical Notes

- Add notes per consultation 
- Use predefined templates for faster input 
- Structured and readable data 


### 3- Appointment Management

- Schedule and track appointments 
- View daily agenda 
- Manage upcoming visits 


### 4- Follow-up & Reminder System

- Set follow-up reminders 
- Identify overdue patients 
- Daily dashboard for pending tasks
